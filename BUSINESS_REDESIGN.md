### 为什么这个架构？

南美和东南亚的共同点不是技术偏好，而是**信仰消费习惯**：
- 巴西人会在圣母像前点蜡烛许愿，泰國人会在四面佛前供花——**为精神安慰付费**是文化常态
- 塔罗在拉美年增长 3.14%，数字化占卜 App 是增速最快品类；泰国神佛市场 150 亿泰铢，52% 年轻人有意愿用 AI 算命
- $0.99 的价格在南美 ≈ 一个 coxinha（巴西小吃）的价格，在东南亚 ≈ 一杯奶茶的 1/3——**几乎没有决策成本**
- $39.99 的水晶手串在南美 ≈ 一件 Zara 衬衫，在东南亚 ≈ 一顿周末聚餐——**合理的信仰投资**

## 四、三大核心模块详细设计

### 模块 A：线上拜神（免费 · DAU 引擎）

**设计目标**：让用户在 60 秒内完成一次有触感的参拜仪式，愿意明天再来。

#### A1. 神祇体系

**第一期上线 8 位核心神（双线）**

南美圣母/圣徒线：

| 神 | 覆盖地区 | 守护领域 |
|----|----------|----------|
| Nossa Senhora Aparecida | 巴西 | 奇迹、母爱、庇护 |
| Virgen de Guadalupe | 墨西哥/中美 | 慈悲、家庭、底层守护 |
| Nuestra Señora de Luján | 阿根廷 | 旅人保护、平安、方向 |
| Santo Niño de Cebú | 菲律宾 | 奇迹、希望、孩子 |

东南亚佛教/印度教线：

| 神 | 覆盖地区 | 守护领域 |
|----|----------|----------|
| 观世音菩萨 Guan Yin | 泰国/华人圈 | 慈悲、救苦、子嗣 |
| 四面佛 Brahma | 泰国 | 全能护佑、事业、财运 |
| 象神 Ganesha | 泰国/印度教区 | 除障、智慧、学业 |
| 天后圣母/妈祖 | 东南亚华人 | 出海平安、女性力量 |

**众筹解锁机制（长尾神祇）**

用户在拜神选择页可以看到一个输入框："你的信仰我们还没有？请告诉我们你想拜哪位神明。"

→ 用户输入后，后端调用 DeepSeek 做神祇知识库模糊匹配
→ 如果匹配到已知但未上线的神 → 展示神名 + 简略介绍 + "已有 {count} 位信徒在等 TA"  + [我也在等，凑满 100 开启参拜] 按钮
→ 如果完全匹配不到 → 用户可提交神名 + 自己的描述，后台人工审核入库
→ 某位神凑满 100 位信徒后 → 触发通知（推送/邮件）→ 该神自动上线，信徒自动切换为首选神

这个机制的作用：
- 尊重用户信仰多样性，不强迫选择
- 把"没有你的神"变成"一起召唤 TA 降临"的社区事件
- 100 人门槛不高，容易触发病毒传播（"帮我一起召唤象神！"）
- 后台数据沉淀：知道用户真正想拜哪些神，指导后续开发优先级

#### A4. 留存机制

- 连续 3 天 → 解锁"长香"交互（火焰更旺，粒子更多）
- 连续 7 天 → 免费深度占卜一次
- 连续 21 天 → 专属守护神头像框 + 神像页面解锁"黄金圣光"特效
- 中断 → 不惩罚，打卡天数保留，只是徽章进度归零
- 每日推送（当地时间 7:00 AM）："{神}已经在等你今天的问候了 🙏"

#### A5. 功德体系（独立成长系统）

**命名**：功德 (Mérito / Merit)

这个概念在天主教"功劳宝库"（Tesouro dos Méritos）和佛教"积累功德"中天然存在，不违和。葡萄牙语 `Mérito`、西班牙语 `Mérito`、英语 `Merit`，三种主要语言一致。

**三条并行的功德之路**

用户不必做所有事才能成长——坚持、传播、供养，三条路一条走到黑都能抵达巅峰。

```
        朝圣者 → 虔信者 → 持光者 → 圣侍 → 近神者
           │        │        │       │       │
时间之路 ──┴────────┴────────┴───────┴───────┘   (每日参拜×年)
传播之路 ──┴────────┴────────┴───────┴───────┘   (邀请新信徒×人)
供养之路 ──┴────────┴────────┴───────┴───────┘   (累计供养×$)
```

用户的三条路径独立计算功德，总功德 = 三条路功德之和。哪条路走得深、哪条路走得浅，在个人中心可视化展示。

---

**时间之路 · 每日参拜**

| 条件 | 累积功德 |
|------|----------|
| 每日参拜 | +1 / 天 |
| 连续 7 天不中断 | +3 额外 |
| 连续 30 天不中断 | +15 额外 |
| 连续 365 天不中断 | +200 额外 |
| 连续 1000 天不中断（≈3 年） | +1000 额外 |
| 累计参拜 3650 天（≈10 年） | +5000 额外 |
| 累计参拜 18250 天（≈50 年） | +30000 直接晋升近神者 |

中断不惩罚——连续奖励只记连续天数，中断后连续计数归零重来，但积累的总功德不丢。

**传播之路 · 引入新信徒**

| 条件 | 累积功德 |
|------|----------|
| 分享链接被点击 | +1 / 次（日上限 5） |
| 被分享者完成首次参拜 | +10 / 人 |
| 被分享者完成首次付费占卜 | +50 / 人 |
| 被分享者购买水晶 | +100 / 人 |
| 被分享者成为虔信者（100 功德） | +200 / 人 |
| 成功召唤新神上线（凑满 100 信徒） | +500 |
| 累计引入 100 万新信徒 | +50000 直接晋升近神者 |

传播链的功德被追踪：每个受邀用户永久绑定推荐人。推荐人在被推荐人获得功德时不会重复获利（避免金字塔），但被推荐人每升一级，推荐人一次性获得阶梯奖励。

**每阶徽章与独特形象**

**1 阶 · 虔信者 Devoto**

徽章：一双手合十，掌心透出暖金色光芒。
形象：旅人跪地，双手捧起一颗发光的星。斗篷边缘开始染上金线。
视觉语言：虔诚、专注、光从手中来。

视觉语言：持灯而行、照亮他人、光芒初成。

#### C2. 水晶品类（5 款核心 SKU — 纯五行体系）

水晶推荐完全基于五行补能逻辑，不绑定特定神祇。

| 五行 | 水晶 | 补益 | 塔罗触发场景 |
|------|------|------|-------------|
| 木 | 绿幽灵 | 生长、突破、事业上升 | 牌阵中权杖牌为主（火），木生火——补木以持续燃烧 |
| 火 | 红玛瑙 | 热情、勇气、行动力 | 牌阵中圣杯牌为主（水），火暖水——补火以驱散情绪寒滞 |

| 金 | 白水晶 | 净化、决断、智慧 | 牌阵中宝剑牌为主（风/思维），金克木之散——补金以聚神 |
| 水 | 黑曜石 | 保护、辟邪、内在平静 | 牌阵中逆位牌多，需水来化解阻滞、以柔克刚 |

五行补能判断由 AI 结合牌阵元素构成 + 用户当前状态综合生成。用户看到的不是"你缺木"，而是"你的能量此刻需要一点生长的力量"。
